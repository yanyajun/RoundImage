# RoundImage
加载网络图片，本地图片，可通过dp值设置圆角大小。
用法：
<com.example.administrator.roundimage.RoundImageView
        android:id="@+id/image"
        android:layout_width="200dp"
        android:layout_height="275dp"
        android:background="@drawable/load_detail"
        app:x_radius="5dp"
        app:y_radius="5dp"/>
